# 🍽️ Restaurant Website

A full-stack restaurant website built using MERN Stack.

## 🚀 Technologies Used

### Frontend
- React.js
- Bootstrap
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## ✨ Features

- User Registration & Login
- Restaurant Menu
- Add to Cart
- Order Management
- QR Payment
- Contact Form
- Admin Dashboard
- Customer Messages

  ## ▶️ Run Project

Frontend:

```bash
cd Restraurant
npm install
npm run dev

Backend:

cd server
npm install
node index.js
